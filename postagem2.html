Subversion

SVN é um sistema de controle de versão OpenSoucure, ele foi desenvolvido para ambientes multiusuários onde existem documentos editados por varias pessoas.Cada alteração que um usuário faz o SVN registra essas alterações e armazene em um banco de dados somente essas alterações feita no documento.Tendo investe que essas modificações tende a ser incrementada por exemplo se já existem três alterações no documento quando próxima pessoa fizer mais uma alteração essa passara a ser a de número quatro e assim sucessivamente. O resultado e que sempre se saber quem alterou, quando alterou e porque alterou e só o que foi alterado fica armazenado no banco de dados.


Vantagens da utilização desse sistema

Saber quem alterou, porque alterou e quando alterou
Compara alterações diferentes (diff)
Volta para versão anterior sem perder a atual
Backup automático

Usos comuns de SVN

Esse sistema foi criar para o desenvolvimento de software e é utilizado na internet por sites com muito movimento como o Google code, SoucerForge, Apache ,KDE, Gnome. Ele também é muito utilizado na manutenção de websitesn é gerencia qualquer tipo de documento. A sua utilização é muito simples pois se a semelha muito como um gerenciamento de arquivos e se uso se da através de uma interface gráfica simples. Exemplo de ferramentas como tortoise.
